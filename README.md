# Portfolio Bima

A modern, responsive personal portfolio website built with **React** and **Vite**. This project showcases a clean dark-themed interface inspired by modern developer portfolios, featuring smooth animations, responsive layouts, and an elegant user experience.

![Portfolio Preview](./public/Foto-Utama.png)

---

## ✨ Features

- 🎨 Modern dark UI inspired by minimalist portfolios
- 📱 Fully responsive (Desktop, Tablet, Mobile)
- ⚡ Built with React + Vite
- 🎭 Smooth animations and hover effects
- 🖼️ Hero section with personal image
- 🧩 Clean and reusable React components
- 🚀 Optimized performance
- 🌐 Easy deployment to GitHub Pages or Vercel

---

## 🛠️ Tech Stack

- React
- Vite
- JavaScript (ES6+)

## 🚀 Getting Started

Clone this repository

```bash
git clone https://github.com/bimaaldi/portfolio-bima.git
```

Move into the project directory

```bash
cd portfolio-bima
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Open your browser

```
http://localhost:5173
```

---

## 📦 Build for Production

Generate the production build

```bash
npm run build
```

Preview the production build locally

```bash
npm run preview
```
